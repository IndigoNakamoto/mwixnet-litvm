# Contributing to MLN Stack

## Development Workflow

- All changes must respect the layer boundaries and canonical files defined in `AGENTS.md`.
- Cursor agents are configured with rules and skills in `.cursor/`. See `docs/AGENT_HANDOFF.md` for how to collaborate between Cursor and external models (Grok, Gemini, etc.).
